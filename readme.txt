== Description ==

Mevlânâ Celâleddin Rumî'nin altı ciltlik Farsça eseri, Mesnevî-i Manevî'den seçme beyitleri Türkçe olarak gösteren bir eklenti.

== Installation ==

1. mesnevi-i-manevi klasörünü wp-content/plugins klasörüne yükleyiniz.
2. Eklentiler sayfasından eklentiyi etkinleştiriniz.
3. Görünüm/Bileşenler sayfasından Mesnevi-i Manevi bileşenini istediğiniz bir alana sürükleyip bırakın.
4. Ayrıca yazı ve sayfalarınızda [mesnevi-i-manevi] etiketini ekleyerek de beyitleri gösterebilirsiniz.
